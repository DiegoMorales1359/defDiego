BOOKS?

That's incredible what you can learn in books.

This is a few books recommendations that really improved my knowledge about the Web, Python, Django, CSS... Just have a look if you're interested in improving your skills.

Python

  • Python Crash Course:This book was my first approach in my way to learn Python, it's all what you need if you know nothing about the language.
  • Python Basics:A book written by Dan Bader, (My favorite Pythonista) the most complete guide of all the basic stuff that you need to work with Python.
  • Python Tricks:Another book by Dan Bader, in this one Dan talks about all that little tricks that can make your Python life easier, perfect if you feel confident about your Python skills.

Django

  • Django Crash Course:Starting in your django path? An excellent way to begin is with this crash course, step by step this couple, Daniel and Audrey, explain a complete Django project.
  • Django for beginners:This one is the bestseller Django books and you can see clearly why, my personal pocket guide when I'm stuck in some part on my Django project.
  • Two Scoops of Django:If you ask somebody in the Django subreddit what is the best book to polish your Django skills, one and only one will be the answer, this book.
